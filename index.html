Transport-neutral agent messaging.

Agent Mail is a structured envelope protocol for reliable agent-to-agent communication — with delivery state tracking, receipt evidence, and an MCP server interface that works with any Claude agent runtime.

Message lifecycle

Sender Transport (Discord) Receiver │ │ │ ├─ agent_mail.send ────────────────┤ │ │ creates envelope + ledger │ │ │ │ │ ├─ posts to Discord ───────────────▶ │ │ │ │ ├─ agent_mail.state.update(sent) ──┤ │ │ records channel_id+message_id │ │ │ ├─ receive_inbound ───────▶ │ │ parse → validate │ │ │ idempotency check │ │ │ store + delivered │ │ │ │ │ ◀──────────── ACK reply ────────┼─────────────────────────┤ │ │ │ └─ agent_mail.state.update(acknowledged) │ records ACK message_id │ │

MCP tools

agent_mail.sendCreate and queue an outbound envelope
agent_mail.replyReply within an existing conversation
agent_mail.state.updateRecord a state transition with transport evidence
agent_mail.conversation.getQuery full conversation history and ledger
agent_mail.conversation.listList conversations by status
agent_mail.mailbox.capabilitiesReturn the local mailbox capability advertisement
agent_mail.route.discoverResolve a mailbox alias to candidate routes and capabilities
agent_mail.bridge.authority.listList bridge actors authorized to assert state transitions
agent_mail.signature.canonical_bodyProduce the deterministic signing body for an envelope
